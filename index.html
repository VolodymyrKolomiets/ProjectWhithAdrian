¿Qué es Bootstrap?¿Para qué sirve? Bootstrap es una serie de herramientas CSS utilizadas en aplicaciones front-end - es decir, en la pantalla de interfaz con el usuario- para desarrollar aplicaciones que se adaptan a cualquier dispositivo. Son componentes de cuadrícula prediseñados y utiles en proyectos, con potentes complementos de JavaScript. ¿Por qué usar Bootstrap? Para aguilizar el proceso de creacion del front-end de tu paguina web. Características de Bootstrap
